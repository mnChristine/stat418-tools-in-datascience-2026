# Week 2: Data Science in the Command Line

## Week Overview

This week we dive into the Unix command line and bash scripting. The command line is the foundation of modern data science workflows - it's how you interact with servers, automate tasks, process data, and chain together tools. We'll learn the Unix philosophy of small, composable tools that do one thing well, and how to write bash scripts that automate repetitive tasks.
