# Week 3: Git Workflows & Containerization

## Week Overview

This week we dive deeper into Git workflows for collaboration and introduce containerization for reproducible environments. You'll learn how to work effectively with teams using Git, handle merge conflicts, and package your applications with Podman so they run the same way on any computer.
